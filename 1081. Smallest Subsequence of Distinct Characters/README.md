
<h3>Smallest Subsequence of Distinct Characters</h3>
<div class="HTMLContent_html__0OZLp" data-track-load="description_content"><p>Given a string <code>s</code>, return <em>the </em><span class="cursor-pointer relative text-dark-blue-s text-sm" data-keyword="lexicographically-smaller-string"><button aria-controls="radix-_r_25_" aria-expanded="false" aria-haspopup="dialog" class="" data-state="closed" type="button"><em>lexicographically smallest</em></button></span> <span class="cursor-pointer relative text-dark-blue-s text-sm" data-keyword="subsequence-string"><button aria-controls="radix-_r_26_" aria-expanded="false" aria-haspopup="dialog" class="" data-state="closed" type="button"><em>subsequence</em></button></span><em> of</em> <code>s</code> <em>that contains all the distinct characters of</em> <code>s</code> <em>exactly once</em>.</p>
<p> </p>
<p><strong>Example 1:</strong></p>
<pre><strong>Input:</strong> s = "bcabc"
<strong>Output:</strong> "abc"
</pre>
<p><strong>Example 2:</strong></p>
<pre><strong>Input:</strong> s = "cbacdcbc"
<strong>Output:</strong> "acdb"
</pre>
<p> </p>
<p><strong>Constraints:</strong></p>
<ul>
<li><code>1 &lt;= s.length &lt;= 1000</code></li>
<li><code>s</code> consists of lowercase English letters.</li>
</ul>
